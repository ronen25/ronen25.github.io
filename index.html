rpiweatherd / rpiweatherd-qt

rpiweatherd is a daemon for the Raspberry Pi that allows monitoring and collection of weather data. It supports multiple sensors, and exports the data in a convinient RESTful API, returning the data as JSON, using a custom-built HTTP server. The data itself is stored in a SQLite database, so querying it manually is also possible - either with the provided Python scripts, or with built-in SQLite utilities.

rpiweatherd-qt is a desktop client for rpiweatherd written in C++11 + Qt 5 Widgets.
